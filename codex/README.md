# Codex Workspace

This folder is the isolated workspace for Codex-owned testing and verification work.

## Layout
- `notes/` exploratory notes
- `plans/` phased plans and execution outlines
- `reports/` bug reports and verification summaries
- `skills/` Codex-only operational playbooks
- `artifacts/` generated outputs from test runs
- `logs/` run logs for reproducibility

## Guardrails
- Keep existing project testing assets intact.
- Build new automation layers here first, then integrate intentionally.
- Prefer reproducible, evidence-backed outputs.
